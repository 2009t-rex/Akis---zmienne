Stylowanie thumb — ::-webkit-slider-thumb

Thumb to ruchome kółko, które przeciągasz na suwaku. Możemy zmieniać jego kolor, rozmiar, kształt oraz dodać mu cień. Styluje się go za pomocą pseudo-elementu: 

input[type="range"]::-webkit-slider-thumb {
    -webkit-appearance: none;
    appearance: none;

    width: 26px;
    height: 26px;
    background: #333;
    border-radius: 50%;
    cursor: pointer;
}

Kolorowa skala — gradient pod suwakiem

Pasek range może mieć kolorową skalę – np. czerwony → zielony. Robimy to ustawiając gradient jako tło toru suwaka. 

input[type="range"]::-webkit-slider-runnable-track {
    height: 12px;
    border-radius: 6px;
    background: linear-gradient(to right, red, orange, yellow, limegreen, green);
}

Tooltip pokazujący wartość

Tooltip to mały dymek nad suwakiem pokazujący aktualną wartość. Pozycja tooltipa zmienia się w zależności od położenia thumb.

HTML


    
    50

CSS

.range-box {
    position: relative;
    width: 350px;
}

#tooltip {
    position: absolute;
    top: -35px;
    background: #333;
    color: white;
    padding: 5px 10px;
    border-radius: 6px;
}

JavaScript

const r = document.getElementById("r");
const tooltip = document.getElementById("tooltip");

r.addEventListener("input", () => {
    tooltip.textContent = r.value;

    const percent = (r.value - r.min) / (r.max - r.min);
    tooltip.style.left = `calc(${percent * 100}% )`;
});

Rozszerzenie<dialog>

Powtórka - znacznik <dialog> pozwala tworzyć wyskakujące okna (modalne lub niestandardowe). Możemy je otwierać, zamykać i stylować. Działa dobrze w nowoczesnych przeglądarkach.

HTML

<dialog id="myDialog">
    <p>To jest przykładowe okno dialogowe.</p>
    <button id="closeDialog">Zamknij</button>
</dialog>

<button id="openDialog">Otwórz dialog</button>

CSS

dialog {
    border: 2px solid #333;
    border-radius: 10px;
    padding: 20px;
    width: 300px;
}
button {
    padding: 10px 20px;
    cursor: pointer;
}

JavaScript

const myDialog = document.getElementById("myDialog");
const openBtn = document.getElementById("openDialog");
const closeBtn = document.getElementById("closeDialog");

openBtn.addEventListener("click", () => {
    myDialog.showModal(); // otwiera dialog modalny
});

closeBtn.addEventListener("click", () => {
    myDialog.close(); // zamyka dialog
});
