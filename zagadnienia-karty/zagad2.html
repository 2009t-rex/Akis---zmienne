Działanie content w CSS

Content w CSS służy głównie do wstawiania generowanej treści przed lub po elementach HTML, ale działa tylko w połączeniu z pseudoelementami ::before i ::after.

Content:

  • Pozwala dodać tekst, symbole, obrazy (np. url(...)) bez zmieniania HTML.
  • Często używana do dekoracji, ikon, cytatów, ozdobnych elementów interfejsu.

Przykład

Lista zakupów:

Chleb

Masło

Kurczak

Jajka

Ser

Ogórki

Kod

HTML:

<h2>Lista zakupów: </h2>
<h3>Chleb</h3>
<h3>Masło</h3>
<h3>Kurczak</h3>
<h3>Jajka</h3>
<h3>Ser</h3>
<h3>Ogórki</h3>

CSS:

h3::before{
content: "➡️";
}
