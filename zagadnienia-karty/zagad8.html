Zmienne CSS w JavaScript i Media Queries

Zmienne CSS można w prosty sposób odczytywać i modyfikować za pomocą JavaScript. Umożliwia to dynamiczną zmianę wyglądu strony.

Pobieranie zmiennych do JS i zmiana ich wartości

Przykład kodu:

let root = document.querySelector(":root");
root.style.setProperty('--primary-bg-color', 'green');

W powyższym przykładzie pobieramy element :root i za pomocą setProperty() ustawiamy nową wartość zmiennej — dzięki temu cały wygląd zależny od tej zmiennej natychmiast się aktualizuje.

Zmienna zależna od rozmiaru ekranu

Zmienne CSS współpracują z media queries. Dzięki temu możemy zmieniać wartości zmiennych w zależności od wielkości ekranu. 

:root {
    --primary-bg-color: blue;
}

body {
    background: var(--primary-bg-color);
}

.box {
    --fontsize: 20px;
    font-size: var(--fontsize);
}

@media screen and (min-width: 1600px) {
    :root {
        --primary-bg-color: red;
    }

    .box {
        --fontsize: 50px;
    }
}

Jeśli okno ma mniej niż 1600px, tło jest niebieskie, a czcionka ma 20px. Jeśli jest większe — tło zmienia się na czerwone, a czcionka rośnie do 50px.
