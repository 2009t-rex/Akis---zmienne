Zmienne

Zmienne w CSS, tak jak w innych językach, które znamy mają zasięg globalny i lokalny. Do zmiennych globalnych można się odwołać w całym dokumencie, a do lokalnych tylko w danym selektorze, gdzie ją zadeklarowaliśmy.

Zmienne globalne tworzymy w środku selektora :root - pseudo klasa połączona z elementem głównym dokumentu strony. (w HTML to zawsze znacznik <html>) Przykład zapisu: 

    :root{
        --przyklad-zmiennej-globalnej: wartosc;
    }

Zmienne lokalne tworzymy w selektorze, gdzie będzie ona użyta: 

    p{
        --przyklad-zmiennej-lokalnej: wartosc;
    }

Ważne!

Nazwa zmiennej musi zaczynać się od -- w innym wypadku nie zadziała.
Nazwa jest wrażliwa na wielkość liter.

Przykład tworzenia zmiennych

    :root {
        --primary-bg-color: green; 
    }

    .note {
        --note-bg: yellow; 
    }

Funkcja var()

Służy do pobierania wartości zmiennej CSS i podstawiania jej w danym miejscu stylu. Dzięki temu możemy zarządzać wieloma elementami jednocześnie poprzez zmianę jednej wartości.

Składnia funkcji wygląda następująco: 

var(nazwa-zmiennej, [wartość-zapasowa])

nazwa-zmiennej — wymagana. Nazwa zmiennej, zawsze poprzedzona dwoma myślnikami.
wartość-zapasowa — opcjonalna. Używana tylko jeśli zmienna nie istnieje lub jest niepoprawna.

Zalety korzystania z funkcji var():

  • kod jest łatwiejszy do aktualizacji (zmiana jednego miejsca — efekt w wielu elementach);
  • kod jest czytelniejszy, bo wartości mają swoje opisowe nazwy;
  • łatwiejsze tworzenie wariantów kolorystycznych lub motywów;
  • zmienne lokalne mogą nadpisywać globalne, co pozwala tworzyć wyjątkowe przypadki.

Przykład użycia: 

:root{
    --kolor-glowny: #0a84ff;
}

.card {
    background-color: var(--kolor-glowny);
}

/* lokalne nadpisanie zmiennej */
.card--ciemna {
    --kolor-glowny: #003f70;
    background-color: var(--kolor-glowny);
}

W powyższym przykładzie wszystkie elementy .card używają zmiennej --kolor-glowny zdefiniowanej globalnie, ale element z klasą .card--ciemna deklaruje tę samą zmienną lokalnie, więc jej wartość nadpisuje tę globalną — i tylko ten jeden element ma ciemniejszy kolor.
