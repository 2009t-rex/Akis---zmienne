<address> — dane kontaktowe autora

Znacznik <address> służy do podawania danych autora lub kontaktowych. Automatycznie dodaje odstęp przed i po (jakby dodawał ukryte <br>).

Strona wykonana przez:
Jan Kowalski
Email: jan.kowalski@example.com
Miasto: Warszawa
<details> i <summary> — rozwijane sekcje bez JS

Znaczniki <details> oraz <summary> tworzą rozwijane, interaktywne sekcje bez potrzeby używania JavaScriptu.

Jak to działa?

  • <summary> jest nagłówkiem, który można kliknąć.
  • Po kliknięciu rozwijana jest zawartość <details>.
  • Można dodać atrybut open, aby sekcja była rozwinięta od startu.

Przykład

<details>
    <summary>Kliknij, aby rozwinąć</summary>
    <p>To jest zawartość sekcji.</p>
</details>

Elementy te posiadają natywną obsługę dostępności (klawiatura, aria-expanded), więc są dobrym wyborem zamiast własnych skryptów do rozwijania treści.

<hgroup> — grupa nagłówków (wycofany)

<hgroup> miał służyć do grupowania nagłówków (np. tytułu i podtytułu) jako jednej logicznej jednostki. Obecnie jest wycofany z obecnego standardu W3C HTML5 i nie powinien być używany.

Alternatywa

<header>
    <h1>Tytuł artykułu</h1>
    <p>Podtytuł</p>
</header>
<mark> — wyróżnianie tekstu

<mark> podświetla fragment tekstu — zwykle żółtym tłem. Oznacza coś istotnego lub znalezione słowo (np. podczas wyszukiwania).

Domyślne wartości CSS

mark {
  background-color: yellow;
  color: black;
}

Przykład

<p>To jest <mark>ważne</mark> słowo.</p>
<noscript> — dla użytkowników bez JavaScriptu

<noscript> wyświetla treść tylko wtedy, gdy JavaScript jest wyłączony lub niedostępny.

Przykład

<noscript>
    <p>Musisz włączyć JavaScript, aby korzystać z tej strony.</p>
</noscript>
<svg> — grafika wektorowa

<svg> służy do rysowania grafiki wektorowej bez utraty jakości. Idealne do ikon, ilustracji, wykresów.

Przykład

<svg width="100" height="100" viewBox="0 0 24 24">
    <circle cx="12" cy="12" r="10" fill="red" />
</svg>
<output> — wynik obliczeń

<output> służy do wyświetlania wyników — zwykle z formularzy lub obliczeń dynamicznych (zwykle z JS).

Prosty przykład

<label>A: <input id="a" type="number" value="2"></label>
<label>B: <input id="b" type="number" value="3"></label>
<output id="sum" for="a b">5</output>
<dialog> — okno dialogowe

Znacznik <dialog> służy do tworzenia natywnych, wbudowanych w HTML okien dialogowych (modalnych i niemetalnych). Można je otwierać i zamykać za pomocą JavaScriptu.

Przykład podstawowy

<dialog id="okno">
    <p>To jest okno dialogowe.</p>
    <button onclick="okno.close()">Zamknij</button>
</dialog>

<button onclick="okno.showModal()">Otwórz okno</button>

showModal() — otwiera okno jako modal
close() — zamyka okno
