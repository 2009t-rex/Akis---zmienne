Data-* w CSS i JavaScript

Atrybuty data-* można wykorzystywać nie tylko do przechowywania danych, ale także do stylowania elementów w CSS oraz odczytywania ich w JavaScript.

Data-* w CSS

Do atrybutu data-* odwołujemy się tak samo jak do każdego innego atrybutu. W CSS możemy wykorzystać funkcję attr(), aby wstawić wartość atrybutu np. do pseudoelementu ::after.

Przykład:

p::after {
    content: " to " attr(data-marka-auta) ".";
}

W tym przykładzie po każdym akapicie pojawi się automatycznie informacja, jakie kto ma auto — wszystko pobrane z atrybutu data-marka-auta.

Data-* w HTML

<p data-marka-auta="Volvo">Auto Mateusza</p>
<p data-marka-auta="BMW">Auto Bartka</p>
<p data-marka-auta="Tesla">Auto Tomka</p>

Dodajemy kolejny element za pomocą JavaScript

Teraz dodamy nową osobę — Kamila — który ma Bugatti. Nie musimy zmieniać HTML. Zrobimy to dynamicznie za pomocą JavaScript. 

let paragrafKamila = document.createElement("p");
paragrafKamila.textContent = "Auto Kamila";
paragrafKamila.dataset.markaAuta = "Bugatti";

document.body.appendChild(paragrafKamila);

W ten sposób stworzyliśmy nowy element, ustawiliśmy mu atrybut data-marka-auta i dodaliśmy go na stronę — a CSS automatycznie dopisał informacje po akapicie.
