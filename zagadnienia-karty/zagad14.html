Nowe/ciekawe tagi i style.



Accent-color

Accent-color w CSS służy do zmiany domyślnego koloru elementów formularza, takich jak: checkbox, radio, range i progress. Ułatwia szybkie dopasowanie ich wyglądu do motywu strony bez tworzenia własnych, skomplikowanych styli.

Domyślna kolorystyka:

Nie domyślna kolorystyka:

kod

.checkbox {
accent-color: red;
}
.radio {
accent-color: green;
}
.range {
accent-color: blue;
}

Color-scheme

Dzięki color-scheme przeglądarka sama dopasuje domyślne elementy UI (np. formularze, scroll-bar, tła systemowe) do właściwego motywu, np.:

  • color-scheme: dark; - ustawia na stronie tryb ciemny.
  • color-scheme: light dark; - ustawia tryb automatycznie.



Backdrop-filter

Backdrop-filter w CSS dodaje efekty graficzne na to, co znajduje się za elementem, np. rozmycie, przyciemnienie lub nasycenie. Najczęściej używa się go do efektu szklanego tła (glass effect). Działa tylko gdy element ma przeźroczystość (np. rgba lub opacity).

Kod

.backdrop{
background-color: rgba(255, 255, 255, 0.4);
-webkit-backdrop-filter: blur(5px);
backdrop-filter: blur(5px);
}

Mask

W CSS mask pozwala ukrywać lub pokazywać fragmenty elementu według kształtu maski (np. gradient lub obrazek). Maska działa jak szablon. Używa się jej do efektów wycinania kształtów bez edytowania obrazów.

Kod

.mask {
mask: url(zsl.png) no-repeat 50% 50%;
}

50% 50% oznacza, że maskę przycinamy ze środka obrazka.
